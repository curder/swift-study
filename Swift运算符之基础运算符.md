## Swift运算符之基础运算符

### Swift运算符之赋值运算符

　　赋值运算符应该是很多编程语言中都常见的一种运算符，例如 `var a = 2`，将 2 赋值给 变量 a ，但是在其他大多数语言中，赋值给变量 a 的同时也会有返回值，返回 2 ，此时可以把这个赋值过程作为 `if` 判断的条件。

> **注意：** 这种写法在 Swift 中是不被允许的。

```
var a = 2
if a = 2 { // use of '=' in a boolean context, did you mean '=='?
    
}
```

### Swift运算符之数学运算符

|  运算符  |  举例  |  作用  |
|--|--|--|
| `+` | `a + b 或 +a` |  |
| `-` | `a - b 或 -a` |  |
| `*` | `a * b` |  |
| `/` | `a / b` |  |
| `%` |  `a % b`  |  |
| `++` | `a++ 或 ++a` |  |
| `--` | `a-- 或 --a` |  |
| `+=` | `a += 2` |  |
| `-=` | `a -= 2` |  |
| `*=` | `a *= 2` |  |
| `/=` | `a /= 2` |  |
| `%=` | `a %= 2` |  |
