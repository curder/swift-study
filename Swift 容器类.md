## Swift 容器类 Collections

所谓的容器类，就是程序中的数据结构，来承载我们app中的数据。

### 数组 Array
* 有序的数据序列

例如：`var numbers = [1, 2, 3, 4, 5]` 或者 `var vowels = ["A", "B", "C", "D", "E"]`

### 字典 Dictionary
* 无序
* 唯一性
* 快速查找
* 提供集合操作

例如：`var dir1 = ["key1":"val", "key2":"val2"]`

### 集合 Set
* 键 -> 值数据对

例如：`var set1 = [1, 2, 3]`
